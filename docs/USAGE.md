# Usage Guide

This guide provides information on how to use the e-Dastavej system.

## User Roles
The system supports different user roles with varying levels of access:
- Public
- Clerk
- Staff
- Assistant
- Manager
- Admin

## Features

### 1. Upload Files
To upload a file, navigate to the "Upload File" page, fill in the required fields, and submit the form.

### 2. Search Files
To search for files, navigate to the "Search Files" page, enter your search criteria, and submit the form. You can filter files by type, user role, date range, category, subcategory, and description.

### 3. User and Role Management
Admins can manage users and roles from the Admin Dashboard. They can create new users, assign roles, and create new roles.

### 4. Category and Subcategory Management
Admins can manage categories and subcategories from the Admin Dashboard. They can create and modify categories and subcategories.

## Admin Dashboard
The Admin Dashboard provides an overview of the system, including the latest uploaded files, user and role management, and category management.

## Accessing the Application
Open your web browser and navigate to:
```
http://localhost:8080/e-Dastavej
```

## Login
To log in, enter your username and password on the login page and submit the form. If you don't have an account, contact an admin to create one for you.

## Example
Here is an example of how to upload a file:
1. Navigate to the "Upload File" page.
2. Select a file to upload.
3. Fill in the description, category, and subcategory fields.
4. Click the "Upload" button.

The file will be uploaded, and a confirmation message will be displayed.
